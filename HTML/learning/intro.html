Introduction

HTML (HyperText Markup Language) is the core language used to structure and display content on the web, defining elements like text, images, links, and page layout so browsers can render websites properly.

Ordered lists

  1. Provides the basic structure for all web pages using opening and closing tags.
  2. Supports text, images, links, lists, tables, forms, and multimedia elements such as audio and video.
  3. Works with CSS for styling and JavaScript for interactivity.
  4. Uses attributes like src, href, and alt to add extra information and control element behavior.
